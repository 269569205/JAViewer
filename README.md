## 关于旧版收藏夹无法打开的问题
    有小朋友可能发现了，用旧版收藏的影片在新版无法打开，这是因为影片的网址改变了，需要手动替换旧网址。
### 解决方法：
    用带有查找替换功能的 APP 打开 properties.json 文件，查找 avmoo.asia 并全部替换为 avmask.com，然后重启应用即可。
