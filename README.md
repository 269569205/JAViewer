## 常见问题
#### Q1. 界面白屏（由于在线网站的网址变更，也可能是梯子的问题）
A1. 切换无码再切回有码，应用将自动更新网址。搭梯子/换线路，有码的网站被墙，所以要搭梯子。
#### Q2. 无法打开旧收藏（由于在线网站的网址变更，而旧收藏的网址不会同步更新，导致无法打开收藏）
A2. 用带有文本查找替换功能的应用（比如MT管理器）打开 properties.json 文件（通常在根目录 JAViewer 文件夹内），查找 avmoo.asia 全部替换为 avmask.com 并保存，然后重启应用。**以后网址再改变的话，也是同样操作，我会在此处标注需要替换的内容。**
> 嫌麻烦的话，删掉旧收藏再重新收藏也行。

`avsox.asia > avsox.host` 2020/1/26

`avmoo.asia > avmask.com` 2019/9/19
